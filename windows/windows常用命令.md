# **更改系统区域设置**
*   Windows 8.1/Windows 10/Windows Server 2012/Windows Server 2012 R2/Windows 2016/Windows Server 2019：单击屏幕左下角的Windows徽标，单击k控制面板，然后单击区域。在区域对话框中，单击管理选项卡，然后单击非 Unicode 程序的语言下的**更改系统区域设置。**
